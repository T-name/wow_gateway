进步GM网关
===
开过服的都知道，游戏管理是一个头疼的事情，于是本人开发了一个方便GM管理游戏的工具网关。
本程序仅用于学习交流！！！请勿用于非法用途！！！

**简介**：
    
    这是一款可以方便GM管理游戏的工具！！
    只需要简单配，注册服务账号即可在线使用！
    如果你觉得有意思，不妨往下看！！！

**功能**：

    概览：
        游戏账户
        在线用户
        所有角色
        账户角色
        所有物品
        角色物品

    控制：
        发送物品
        发送金币
        根据角色封禁账户
        聚合补发
        发布游戏公告
        设置GM等级
